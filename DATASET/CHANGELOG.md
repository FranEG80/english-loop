# Changelog

Todos los cambios relevantes del dataset se documentan aquí. El versionado sigue
SemVer: los cambios incompatibles de esquema incrementan la versión mayor; las
ampliaciones compatibles, la menor; y las correcciones editoriales, el parche.

## 0.1.0 — 2026-07-02

- Se crea la estructura independiente de `DATASET/`.
- Se fijan las convenciones editoriales y de publicación.
- Se añaden contratos JSON Schema Draft 2020-12.
- Se registran las fuentes CEFR y los formatos Cambridge B1/B2.
- Se incorpora tooling determinista para validación, evaluación e índices.
