# Resumen

El present perfect passive conecta una acción pasada con una situación presente
y mantiene el foco en el receptor o resultado:

`The application has been approved.`  
`The faulty parts have been replaced.`

Se forma con `have/has been + participio`. `Has` concuerda con un sujeto
singular y `have` con un sujeto plural o con `I/you`.

La pasiva con modales se forma con `modal + be + participio`:

`The form can be completed online.`  
`Safety glasses must be worn.`  
`The figures should be checked again.`

`Been` pertenece al present perfect; `be` aparece después de un modal. Esta
diferencia evita dos errores muy frecuentes: `has be sent` y `must been sent`.

# Objetivos

- Usar `has been` y `have been` según el sujeto.
- Formar negativas y preguntas en present perfect passive.
- Usar `can/can't be + participio` para posibilidad práctica.
- Usar `must/mustn't be + participio` para requisitos y prohibiciones.
- Usar `should be + participio` para recomendaciones.
- Usar `may/might be + participio` para posibilidades.
- Transformar activas sin cambiar el significado ni el tiempo.
- Distinguir `modal + be` de `have/has been`.

# Explicación

## Present perfect passive

Activa:

`The team has repaired the lift.`

Pasiva:

`The lift has been repaired.`

El objeto activo `the lift` pasa a ser sujeto. El auxiliar `has` se mantiene,
se añade `been` y el verbo principal pasa al participio `repaired`.

Con sujeto plural:

`The team has repaired both lifts.`  
`Both lifts have been repaired.`

La concordancia se ajusta al nuevo sujeto: `both lifts have`.

## Pasiva con modales

Activa:

`Visitors can download the map.`  
Pasiva: `The map can be downloaded.`

Activa:

`Staff must lock the gate at dusk.`  
Pasiva: `The gate must be locked at dusk.`

El modal no cambia. Después usamos la forma base `be` y el participio.

# Forma o estructura

## Present perfect passive

- Afirmativa: sujeto + `have/has been + participio`
- Negativa: sujeto + `haven't/hasn't been + participio`
- Pregunta: `Have/Has + sujeto + been + participio?`

`The payment has been received.`  
`The keys haven't been found.`  
`Have all the rooms been inspected?`

Los adverbios habituales mantienen una posición natural:

`The report has already been published.`  
`The parcel hasn't been delivered yet.`  
`Three new stops have recently been added.`

## Pasiva con modales

- Afirmativa: sujeto + `modal + be + participio`
- Negativa: sujeto + `modal + not + be + participio`
- Pregunta: `Modal + sujeto + be + participio?`

`Tickets can be purchased at the machine.`  
`This switch mustn't be touched.`  
`Should the damaged section be replaced?`

# Usos principales

1. **Resultado reciente:** `The missing child has been found.`
2. **Cambio con efecto actual:** `The timetable has been updated.`
3. **Experiencia pasiva:** `The castle has been visited by millions of people.`
4. **Acción aún no completada:** `The final decision hasn't been announced.`
5. **Posibilidad práctica:** `The appointment can be changed online.`
6. **Imposibilidad práctica:** `The file can't be opened without the password.`
7. **Requisito:** `All fields must be completed.`
8. **Prohibición:** `Original documents mustn't be removed.`
9. **Recomendación:** `The batteries should be replaced together.`
10. **Posibilidad incierta:** `The outdoor event may be cancelled.`

# Contrastes importantes

## Has been frente a have been

El auxiliar concuerda con el sujeto pasivo:

`The request has been approved.`  
`The requests have been approved.`

No depende del sujeto de la oración activa original.

## Been frente a be

Present perfect:

`The gate has been locked.`  
Estructura: `has + been + locked`.

Modal:

`The gate must be locked.`  
Estructura: `must + be + locked`.

No usamos `has be locked` ni `must been locked`.

## Pasado simple frente a present perfect passive

`The bridge was closed at 09:15.`  
La hora pasada está terminada y especificada.

`The bridge has been closed because of ice.`  
Importa el resultado actual: ahora está cerrado.

Con marcadores terminados como `yesterday`, `last week` o `in 2022` usamos
normalmente pasado simple, no present perfect.

## Modal activo frente a modal pasivo

`Visitors must leave bags at reception.`  
El sujeto `visitors` realiza la acción.

`Bags must be left at reception.`  
El sujeto `bags` recibe la acción.

## Mustn't be frente a doesn't have to be

`The seal mustn't be broken.`  
Romperlo está prohibido.

`The form doesn't have to be printed.`  
Imprimirlo no es necesario.

La diferencia entre prohibición y ausencia de necesidad se conserva dentro de
la pasiva.

# Participios frecuentes

- `approve → approved`
- `break → broken`
- `choose → chosen`
- `complete → completed`
- `find → found`
- `give → given`
- `hold → held`
- `make → made`
- `pay → paid`
- `publish → published`
- `replace → replaced`
- `send → sent`
- `take → taken`
- `write → written`

La palabra `been` no sustituye al participio principal: `has been written`, no
`has been write`.

# Ejemplos

1. **Your reservation has been confirmed.**
2. **The invoices have already been paid.**
3. **The cause hasn't been identified yet.**
4. **Have the replacement keys been sent?**
5. **The instructions can be downloaded as a PDF.**
6. **This medication must be stored below 25°C.**
7. **The emergency exit mustn't be blocked.**
8. **The final paragraph should be shortened.**
9. **The match may be postponed if the pitch freezes.**
10. **Can the booking be transferred to another person?**

# Errores frecuentes

- ❌ *The booking has be confirmed.*  
  ✅ *The booking has been confirmed.*  
  El perfect necesita `been`.

- ❌ *The documents has been signed.*  
  ✅ *The documents have been signed.*  
  El sujeto plural usa `have`.

- ❌ *The parcel has been send.*  
  ✅ *The parcel has been sent.*  
  Después de `been` usamos participio.

- ❌ *The form can completed online.*  
  ✅ *The form can be completed online.*  
  La pasiva modal necesita `be`.

- ❌ *Helmets must been worn.*  
  ✅ *Helmets must be worn.*  
  Después de un modal usamos `be`, no `been`.

- ❌ *Can be the date changed?*  
  ✅ *Can the date be changed?*  
  El sujeto se coloca entre el modal y `be`.

- ❌ *The result has been announced yesterday.*  
  ✅ *The result was announced yesterday.*  
  Un momento pasado terminado requiere pasado simple.

# Excepciones relevantes

`Should have been + participio` y otros patrones como `might have been sent`
hablan de situaciones anteriores desde una perspectiva modal. Son estructuras
de modal perfect que pertenecen al nivel B2 y no deben confundirse con la
pasiva modal B1 `should be sent`.

`Need to be + participio` y `have to be + participio` no contienen un modal
puro, pero funcionan de forma similar:

`The filter needs to be replaced.`  
`All visitors have to be registered.`

Se incluyen cuando el contexto deja una única respuesta clara.

Algunas frases de present perfect passive describen un estado resultante:
`The door has been locked`. Si necesitamos destacar quién actuó, podemos añadir
`by`, pero normalmente el resultado es la información principal.

La pasiva no elimina la necesidad de usar un verbo transitivo. No decimos `has
been happened`; decimos `has happened`.

# Mini resumen

- Present perfect passive → `have/has been + participio`.
- Sujeto singular → `has been`; plural → `have been`.
- Modal passive → `modal + be + participio`.
- Negativa perfect → `hasn't/haven't been`.
- Negativa modal → `modal + not + be`.
- Pregunta perfect → `Have/Has + sujeto + been ...?`
- Pregunta modal → `Modal + sujeto + be ...?`
- `Yesterday` y fechas terminadas suelen pedir pasado simple.

# Comprobación rápida autocorregible

1. The schedule ___ updated. (`have`, present perfect passive)
   - Respuesta: `has been`
   - Explicación: el sujeto singular usa `has been`.
2. All the seats ___ sold. (`have`, present perfect passive)
   - Respuesta: `have been`
   - Explicación: el sujeto plural usa `have been`.
3. The form can ___ online. (`complete`, pasiva)
   - Respuesta: `be completed`
   - Explicación: después de `can` usamos `be + participio`.
4. Safety glasses must ___ in this area. (`wear`, pasiva)
   - Respuesta: `be worn`
   - Explicación: la pasiva modal requiere `be worn`.
5. ___ the results been published yet?
   - Respuesta: `Have`
   - Explicación: la pregunta plural comienza con `Have`.
