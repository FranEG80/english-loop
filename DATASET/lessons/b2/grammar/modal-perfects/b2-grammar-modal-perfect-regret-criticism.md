# Resumen

`Should have` evalúa una acción pasada que habría sido preferible; `shouldn't
have` critica una acción realizada. `Could have` presenta una posibilidad u
oportunidad que no se aprovechó. `Needn't have` indica que una acción sí se hizo,
pero resultó innecesaria.

# Objetivos

- Deducir por el contexto si la acción ocurrió o no.
- Elegir la estructura que expresa consejo retrospectivo, oportunidad o falta
  de necesidad.
- Utilizar participios irregulares correctos.

# Explicación

`You should have checked the address` implica que no lo comprobaste y que
hacerlo habría evitado un problema. `You shouldn't have shared the link` implica
que sí lo compartiste.

`We could have taken a taxi` presenta una alternativa disponible que no se
eligió. `You needn't have bought any bread` confirma que compraste pan, aunque
no hacía falta. En cambio, `You didn't need to buy bread` solo afirma que no era
necesario; a menudo se entiende que no lo compraste, pero la frase no lo
garantiza por sí sola.

# Forma o estructura

- `should have + participio`: acción recomendable no realizada.
- `shouldn't have + participio`: acción realizada que fue un error.
- `could have + participio`: opción o capacidad pasada no aprovechada.
- `needn't have + participio`: acción realizada pero innecesaria.
- `didn't need to + infinitivo`: ausencia de necesidad, sin afirmar por la forma
  si la acción ocurrió.

# Usos principales

1. Arrepentimiento: `I should have accepted their offer.`
2. Crítica: `You shouldn't have interrupted her.`
3. Alternativa perdida: `We could have used the side entrance.`
4. Esfuerzo innecesario: `You needn't have printed every page.`
5. Consejo retrospectivo: `They should have tested the update first.`

# Contrastes importantes

`Couldn't have` suele expresar imposibilidad: `I couldn't have finished without
your help`. `Could have` afirmativo puede indicar una oportunidad perdida:
`I could have finished earlier`.

`Had to` indica obligación; `needn't have` indica precisamente que no existía
necesidad, aunque la acción se realizó.

# Ejemplos

1. **We should have left more time for questions.**
2. **You shouldn't have posted the address publicly.**
3. **They could have avoided the queue by booking online.**
4. **I needn't have brought a towel; the hotel provided one.**
5. **We didn't need to replace the cable because the old one still worked.**

# Errores frecuentes

- ❌ *You should have went earlier.*  
  ✅ *You should have gone earlier.*  
  Usamos el participio `gone`.

- ❌ *I didn't needed to call.*  
  ✅ *I didn't need to call.*  
  Tras `didn't` usamos infinitivo.

- ❌ *You needn't have to bought lunch.*  
  ✅ *You needn't have bought lunch.*  
  La estructura no incluye `to`.

# Excepciones relevantes

`Could have` no siempre comunica arrepentimiento: en una deducción como `The
error could have originated in the database`, significa posibilidad. El
resultado pragmático depende de si el contexto presenta una alternativa
disponible o una explicación incierta.

# Mini resumen

- No se hizo, pero era recomendable → `should have`.
- Se hizo, pero fue un error → `shouldn't have`.
- Había otra opción → `could have`.
- Se hizo sin necesidad → `needn't have`.

# Comprobación rápida autocorregible

1. You ___ have checked the expiry date before buying it.
   - Respuesta: `should`
   - Explicación: era recomendable, pero no se hizo.
2. We ___ have booked a taxi; the station was only five minutes away.
   - Respuesta: `needn't`
   - Explicación: se reservó, pero resultó innecesario.
3. They ___ have taken the earlier route, but they didn't know about it.
   - Respuesta: `could`
   - Explicación: era una alternativa disponible no aprovechada.
