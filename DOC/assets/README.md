# EnglishLoop asset pack

Generated in one `$imagegen` work block for frontend phase 3.

## Visual direction

- Soft 2D editorial illustration for an adult learning product.
- Clean organic shapes and very light paper texture.
- Deep teal and `#14B8A6`, warm cream, restrained coral, dark navy details.
- No embedded text, wordmark, watermark, SVG, or childlike treatment.
- Loopy uses `mascot/loopy-reference.webp` as the identity master.

## Delivery

- Brand mark and badge variant: WebP with alpha.
- Mascot reference and six poses: 1254 × 1254 WebP with alpha.
- Illustrations: WebP with alpha; their intrinsic dimensions are preserved.
- Activity illustrations: fourteen WebP assets with alpha under
  `public/illustrations/activities/`.
- Landing photography: three opaque 1600 × 900 WebP assets under
  `public/photos/landing/`; `high-five-loopy.webp` combina fotografía
  editorial con la mascota ilustrada.
- Social cover: 1200 × 630 opaque WebP.
- App icons: opaque PNG at 180, 192, and 512 pixels.
- Contact sheet: `contact-sheet.webp`.

The `EnglishLoop` wordmark is intentionally not rasterized. Render it in Geist
beside `public/brand/englishloop-mark.webp`.
